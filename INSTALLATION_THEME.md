# Installation du Thème WordPress Montaiguet (Version Améliorée)

## Téléchargement

Le thème se trouve dans le fichier : **`montaiguet-theme-updated.zip`** (33 KB)

Ce fichier se trouve à la racine du projet.

## Installation sur WordPress

### Méthode 1 : Via l'interface d'administration WordPress (Recommandée)

1. **Téléchargez** le fichier `montaiguet-theme-updated.zip` sur votre ordinateur

2. **Connectez-vous** à votre administration WordPress :
   - URL : `https://miellerie-montaiguet.fr/wp-admin`

3. **Accédez** à la section des thèmes :
   - Allez dans **Apparence** > **Thèmes**

4. **Ajoutez** le nouveau thème :
   - Cliquez sur le bouton **Ajouter**
   - Cliquez sur **Téléverser un thème**

5. **Installez** :
   - Cliquez sur **Choisir un fichier**
   - Sélectionnez le fichier `montaiguet-theme-updated.zip`
   - Cliquez sur **Installer maintenant**

6. **Activez** :
   - Une fois l'installation terminée, cliquez sur **Activer**

### Méthode 2 : Via FTP

1. **Décompressez** le fichier `montaiguet-theme-updated.zip` sur votre ordinateur
   - Vous obtiendrez un dossier `montaiguet-theme`

2. **Connectez-vous** à votre serveur via FTP

3. **Naviguez** vers le dossier des thèmes :
   - `/wp-content/themes/`

4. **Supprimez** l'ancien thème (optionnel, mais recommandé) :
   - Renommez d'abord l'ancien dossier `montaiguet-theme` en `montaiguet-theme-backup`

5. **Uploadez** le nouveau dossier `montaiguet-theme` dans `/wp-content/themes/`

6. **Activez** le thème depuis l'administration WordPress :
   - Allez dans **Apparence** > **Thèmes**
   - Cliquez sur **Activer** sous le thème Montaiguet

## Support

Si vous rencontrez des problèmes :
1. Vérifiez que vous utilisez WordPress 5.0+ et PHP 7.4+
2. Videz le cache de votre site et de votre navigateur
3. Désactivez temporairement les plugins de cache

## Backup

**Important** : Avant d'installer le nouveau thème, il est recommandé de :
- Faire une sauvegarde complète de votre site WordPress
- Ou au minimum, sauvegarder l'ancien thème via FTP
